配置tensorflow

pycharm的tensorflow环境配置

前提：pycharm已经安装好

安装路径：https://www.jetbrains.com/pycharm/download/#section=windows

安装完之后需要重启电脑

环境搭建的步骤：

  1. 安装基本环境
  2. 安装TensorFlow
  3. 验证安装

一、安装基本环境

TensorFlow支撑

  • Window、MacOS、Linux都已支持TensorFlow
  • Window用户，官方最新支撑Python3.7及以上版本
  • 推荐安装Anaconda

Anaconda下载

1.下载地址

https://repo.anaconda.com/archive/

1） MacOS、Linux可下载Python2.7和Python3.3+的任意版本

2）前面均为傻子安装，其中注意这里要勾选添加环境变量

3）检查是否有环境变量，没有的话要添加上去

2.创建anaconda虚拟环境

anaconda安装安装完成以后，默认的都是在base环境下，由于在开发过程中，可能用到的语言环境及版本

都不一样，所以我们单独在anaconda里面创建一个服务于tensorflow的虚拟环境。

  1. 首先我们打开cmd或者anaconda prompt
  2. 输入conda env list 查看当前的虚拟环境，默认只有一个base
  3. 执行conda create -n tensor python=3.7 (tensor为虚拟环境名称，自定义)
  4. 如果出现UnavailableInvalidChannel: The channel is not accessible or is invalid.错误，执行：conda config -- remove-key channels 用来恢复默认源，然后再执行第三步命令，最后输入y确认

二、安装TensorFlow

以下以cpu版本为例

  1. 创建虚拟环境以后，再次执行conda env list查看是否有你自己创建的虚拟环境
  2. 激活/进入到创建的虚拟环境中，执行：conda activate tensor，前面出现一个括号是你的虚拟环境名称即正确
  3. 在当前虚拟环境中，安装tensorflow，：pip install tensorflow==2.9 -i https://pypi.tuna.tsinghua.edu.cn/simple/
  4. 输入conda list,检查是否有tensorflow
  5. 验证tensorflow是否可用
    • 在当前环境输入python
    • 输入import tensorflow as tf
    • 然后做一个简单的计算print(tf.constant(1.)+tf.constant(2.))
    • 检查如果有计算结果代表可用,tensorflow安装成功

三、配置pycharm并验证

在设置里面，我们选择右边添加新的编译器

在interpreter选择你anaconda的安装路径，找到envs文件下对应的虚拟环境名称，选择文件里面的python.exe

新建pyhton文件并输入以下内容运行成功即可
