QQ插件!

QQNT防撤回、壁纸优化插件

这是什么

LiteLoaderQQNT 是 QQNT 的插件加载器，一般在 QQNT 的环境内简称为 LiteLoader
它可以让你自由地为 QQNT 添加各种插件，并实现例如美化主题、增加功能等各种功能

仓库地址：https://github.com/LiteLoaderQQNT/LiteLoaderQQNT

安装教程

注：此教程根据使用文档编写。

1.下载

方法一 Clone

使用 Git 工具将 LiteLoaderQQNT 仓库 Clone 到本地任意位置

方法二 Release

在 LiteLoaderQQNT的Release中 Latest 内，下载 LiteLoaderQQNT.zip 文件，将压缩包内 LiteLoaderQQNT 目录解压到任意位置

2.安装

a.路径分隔符建议使用 /（通用）或使用 \\（仅限 Windows），单独使用 \ 会导致报错

b.您的QQNT目录需要有读写权限

3.修补

此时打开QQ应该是损坏情况，我们要进行修复。

在 QQNTFileVerifyPatch的Release 中 Latest 内，下载 QQNTPatcher.exe 文件，以管理员身份运行将弹出文件选择框，进入 QQNT 安装目录选择 QQ.exe 开始修补，等待输出修补完成即可关闭软件

4.检查

按照上述教程完成安装后，运行 QQNT 并打开设置，查看左侧列表是否出现 LiteLoaderQQNT 选项

如果有显示，即安装成功，玩的开心！

5.下载插件/商店
